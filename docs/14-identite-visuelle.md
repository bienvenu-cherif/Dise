# Identite visuelle CotaDISE

## Objectif

CotaDISE doit avoir une identite claire, institutionnelle et moderne, lisible sur smartphone et coherente avec la Division ISE et l'ENSEA.

## Sources graphiques

Les logos sources sont conserves dans:

- `cotadise-mobile/assets/logos/logo-division.png`;
- `cotadise-mobile/assets/logos/logo-ecole.png`.

Le logo Division sert de base principale pour l'identite de l'application.

Le logo ENSEA sert de reference institutionnelle et pourra etre utilise sur les ecrans d'accueil, pages a propos, documents publics ou supports de communication.

## Couleurs principales

Palette extraite des deux logos:

- Bleu nuit: `#1f1c5b`;
- Bleu ENSEA: `#154a8b`;
- Cyan: `#22a6cf`;
- Ivoire: `#eff2c5`;
- Dore: `#cfbe63`;
- Blanc: `#ffffff`.

## Icone mobile

L'icone actuelle est volontairement simple:

- fond bleu nuit vers cyan;
- inscription centrale `DISE`;
- nom `Cotadise` sous l'inscription principale;
- anneaux et accent dore inspires du logo Division.

Cette approche permet de rester lisible en petite taille sur Android et iOS.

Fichiers generes:

- `cotadise-mobile/assets/icon.png`;
- `cotadise-mobile/assets/adaptive-icon.png`;
- `cotadise-mobile/assets/favicon.png`.

## Ecran de demarrage

L'ecran de demarrage reprend la meme direction graphique que l'icone:

- fond vertical bleu nuit vers cyan;
- marque `DISE`;
- nom `Cotadise`;
- mention `Cotisations ISE`.

Fichier genere:

- `cotadise-mobile/assets/splash.png`.

## Regles de design

- Ne pas surcharger l'icone avec les deux logos complets.
- Garder `DISE` comme signal principal en petite taille.
- Utiliser `Cotadise` comme nom d'application visible.
- Utiliser le logo ENSEA dans des contextes ou il reste lisible.
- Eviter les fonds trop clairs pour les ecrans de marque.
- Conserver un contraste fort pour l'accessibilite.

## Evolutions possibles

- Creer une variante institutionnelle avec le logo ENSEA sur l'ecran a propos.
- Ajouter une page de credits indiquant la Division ISE et l'ENSEA.
- Produire une banniere store Android/iOS avec les deux identites.
- Creer des captures store a partir des principaux ecrans de l'application.

## Interface tresorier

L'interface web du tresorier reprend la meme palette:

- fond bleu nuit vers bleu ENSEA;
- hero clair ivoire pour la lecture rapide;
- accents cyan pour la progression et les actions;
- cartes blanches pour les indicateurs et les tableaux;
- bande de campagne pour resumer l'annee suivie, l'etat de preparation et l'action prioritaire.
- cartes d'intention en haut des onglets sensibles: Etudiants, Paiements, Wave, Classement, Defis, Alumni, Dons et Audit.
- messages de confiance sur les donnees officielles, la tresorerie, les secrets Wave et la tracabilite.
- anneau de progression dans le hero tresorier pour visualiser immediatement le taux de collecte.
- barre de repartition des etudiants soldes, partiels et sans paiement dans le resume tresorier.
