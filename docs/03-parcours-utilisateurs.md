# Parcours utilisateurs

## Parcours etudiant

### Activation du compte nouvel ISE1

1. Le nouvel ISE1 installe l'application mobile.
2. Il choisit l'option d'activation de compte.
3. Il recherche son nom dans la liste officielle importee par le tresorier.
4. Le systeme verifie que le nom correspond a un etudiant invite et non encore active.
5. L'etudiant complete ses informations: email valide, telephone, numero Wave si different, mot de passe.
6. Le systeme envoie si besoin un message de confirmation a l'email.
7. Le compte devient actif.
8. L'application affiche son tableau de bord personnel.

### Connexion normale

1. L'etudiant ouvre l'application mobile.
2. Il saisit son email ou telephone et son mot de passe.
3. Le systeme verifie son compte.
4. L'application affiche son tableau de bord personnel.

### Tableau de bord etudiant

L'etudiant voit:

- Son nom et sa classe.
- Le montant total a cotiser.
- Le montant deja cotise.
- Le reste a payer.
- Le pourcentage de progression.
- Une courbe d'evolution des paiements.
- Son rang dans sa classe.
- Les dernieres alertes.
- Les defis en cours.

### Paiement Wave

1. L'etudiant clique sur payer.
2. Il choisit ou confirme le montant.
3. Il indique s'il paie pour lui-meme ou pour un camarade.
4. L'application verifie que le numero utilise est compatible avec un paiement Wave.
5. L'application initie une transaction Wave.
6. Le systeme attend la confirmation du paiement.
7. Apres confirmation, le paiement est enregistre.
8. La cotisation du beneficiaire est ajustee.
9. La progression, le classement et les alertes sont mis a jour.

### Paiement main a main

1. L'etudiant remet l'argent au tresorier.
2. Le tresorier recherche le compte de l'etudiant.
3. Le tresorier enregistre le paiement main a main.
4. Le systeme ajuste la cotisation de l'etudiant.
5. L'application mobile de l'etudiant affiche immediatement la nouvelle progression.

### Defi entre etudiants

1. L'etudiant choisit un camarade.
2. Il lance un defi: finir sa cotisation avant lui.
3. L'autre etudiant reçoit une notification.
4. Il accepte ou refuse.
5. Si le defi est accepte, le systeme suit la progression des deux.
6. Le premier qui atteint 100% gagne le defi.
7. Les deux reçoivent une notification de resultat.

## Parcours tresorier

### Preparation de l'annee

1. En octobre, le nouveau bureau prend les reignes de la division.
2. Le tresorier cree ou prepare la nouvelle annee academique.
3. Le tresorier ferme les comptes de l'annee precedente.
4. Le systeme prepare les passages automatiques.
5. Le tresorier consulte la liste des changements proposes.
6. Le tresorier corrige les exceptions.
7. Le tresorier valide le passage.

### Import des nouveaux ISE1

1. Le tresorier importe un fichier Excel ou CSV.
2. Le systeme lit les colonnes: prenom, nom, email, telephone.
3. Le systeme cree les comptes ISE1 en statut invite ou profil a completer.
4. Le systeme signale les lignes invalides ou les doublons.
5. Les etudiants non presents dans cette liste ne peuvent pas activer de compte.
6. Les nouveaux ISE1 finalisent eux-memes leur profil dans l'application mobile.

### Fixation des montants

1. En novembre, le tresorier definit le montant ISE1.
2. Le tresorier definit le montant ISE2.
3. Le tresorier definit le montant ISE3.
4. Le tresorier ajoute si besoin des exceptions individuelles.
5. Le systeme calcule les cotisations a generer.

### Generation des cotisations

1. Le tresorier lance la generation.
2. Le systeme cree une cotisation pour chaque etudiant eligible.
3. Le systeme ignore les alumni, abandons et exclus.
4. Le tresorier voit le resume financier attendu.

### Suivi quotidien

Le tresorier voit:

- Total attendu.
- Total collecte.
- Reste a collecter.
- Nombre d'etudiants a jour.
- Nombre d'etudiants en retard.
- Classement par classe.
- Exports Excel.
- Paiements recents.

### Messages aux alumni

1. Le tresorier choisit une promotion sortante.
2. Il redige une demande d'aide ou un message personnalise.
3. Le systeme envoie le message aux alumni de cette promotion.
4. Les alumni peuvent repondre par don, aide ou prise de contact.
